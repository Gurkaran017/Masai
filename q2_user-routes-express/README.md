# Express User Routes App

## Objective
Create an Express server with user-related routes in a single file (`server.js`).

## Routes

- `GET /users/get`  
  Returns a single dummy user.

- `GET /users/list`  
  Returns an array of three dummy users.

- `Any Other Route`  
  Returns a 404 JSON error.
